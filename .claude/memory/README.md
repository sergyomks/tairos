# Memoria Colectiva de Tayros

Aquí se almacenará la memoria persistente del equipo de desarrollo, decisiones de arquitectura y preferencias generales.

## Estructura Esperada:
- **`MEMORY.md`**: Índice y resumen de decisiones clave.
- **`user/`**: Preferencias específicas de los humanos del equipo.
- **`feedback/`**: Registro de correcciones aplicadas a los agentes.
- **`project/`**: Estado y decisiones tomadas en proyectos específicos.
- **`reference/`**: Soluciones y patrones técnicos comunes.
