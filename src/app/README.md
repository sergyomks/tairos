# Next.js App Router (UI de Tairos OS)

Contiene las rutas y vistas del dashboard principal de Tairos OS.

## Páginas Clave a Implementar:
- **`chat/`**: La interfaz de la sala de chat en tiempo real.
- **`mission-control/`**: El panel Bento Grid para visualizar métricas, estado de los servidores y proyectos.
- **`tasks/`**: Vista detallada de la cola y progreso de tareas A2A.
